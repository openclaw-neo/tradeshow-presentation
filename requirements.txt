# Presentation Requirements

Create a professional, modern HTML presentation for a tech tradeshow product recommendations deck.

## Design Style
- **Color palette:** Dark theme (#0a0a0a background, #f8f8f8 text, #3b82f6 accent)
- **Typography:** Inter (body) + Space Grotesk (headings)
- **Layout:** 16:9 aspect ratio, 80px margins, 40px grid
- **Mood:** Modern, clean, professional, tech-forward

## Technical Requirements
- Self-contained HTML with inline CSS
- No external dependencies (except Google Fonts CDN)
- Keyboard navigation (arrow keys)
- Slide counter
- 1920x1080 viewport
- Smooth transitions between slides
- Responsive design (works on smaller screens too)

## Output
- Single HTML file at presentation/slides.html
- All styling inline, no external CSS files
- Include CSS for print (media query for A4)
